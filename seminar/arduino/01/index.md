## arduino IDE のインストール

* https://www.arduino.cc/ にアクセスして、Downloadをクリックします
   
  ![](../img/arduino_ide_install/01.png)
   
  ----
   
* Download the Arduino Software のページで、Windows Installer をクリックします

  ![](../img/arduino_ide_install/02.png)
  ----
 
*  JUST DOWNLOADをクリックして、Installerをダウンロードします

  ![](../img/arduino_ide_install/03.png)
  ----
 
* ダウンロードしたInstallerをダブルクリックして起動します
* 旧バージョンのIDEが既にインストールされている場合は、アンインストールしますので、OKを押します

  ![](../img/arduino_ide_install/04.png)
  ----
 
* アンインストールするファイルの場所を確認して、Uninstallを押します

  ![](../img/arduino_ide_install/05.png)
  ----
 
* 削除の確認画面で、OKを押します

  ![](../img/arduino_ide_install/06.png)
  ----
 
* アンインストール作業の画面を表示します。Uninstall終了後、Closeを押します

  ![](../img/arduino_ide_install/07.png)
  ----
  ![](../img/arduino_ide_install/08.png)
  ----
  ![](../img/arduino_ide_install/09.png)
  ----
 
* arduino IDEをインストールします。I Agreeを押します

  ![](../img/arduino_ide_install/10.png)
  ----

* インストールの設定をします。すべてにチェックが入っているか確認して、Nextを押します

  ![](../img/arduino_ide_install/11.png)
  ----

* インストールするフォルダを確認して、Installを押します

  ![](../img/arduino_ide_install/12.png)
  ----

* インストール作業の画面を表示します。インストール終了後にCloseを押します

  ![](../img/arduino_ide_install/13.png)
  ----
  ![](../img/arduino_ide_install/14.png)
  ----
 
* デスクトップのArduinoのアイコンをダブルクリックして、IDEを起動します

  ![](../img/arduino_ide_install/15.png)
  ----
  ![](../img/arduino_ide_install/16.png)
----
